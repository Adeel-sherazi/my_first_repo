# Source Code

This directory contains the source code for the first project.

## Files

### hello_world.py
A simple Python program demonstrating:
- Functions and parameters
- Docstrings and documentation
- Basic string formatting
- The `if __name__ == "__main__"` pattern

**To run:**
```bash
python3 src/hello_world.py
```

### hello_world.js
A simple JavaScript program demonstrating:
- Functions with default parameters
- JSDoc documentation
- Template literals
- Console output

**To run:**
```bash
node src/hello_world.js
```
